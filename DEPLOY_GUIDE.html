🚀 Pretzel公式サイト 公開手順ガイド

GitHub Pagesを使って、無料でサイトを公開する手順です

📋 やること一覧（所要時間: 約15分）

1

GitHubアカウント作成（すでにあればスキップ）

  • https://github.com にアクセス
  • Sign up をクリック → メールアドレス、パスワードを設定
  • 無料プランでOK（クレジットカード不要）
2

リポジトリ作成

  • 右上の 「+」→「New repository」 をクリック
  • Repository name: pretzel-website
  • Public を選択
  • 「Create repository」 をクリック
💡 ポイント: Publicにしないと GitHub Pages（無料版）が使えません。
3

ファイルアップロード

  • 「uploading an existing file」 リンクをクリック
  • website フォルダの中身を全部ドラッグ＆ドロップ
index.html
services.html
pricing.html
results.html
faq.html
contact.html
launch.html
assets/ css/, js/
  • 「Commit changes」 をクリック
💡 ポイント: website フォルダ自体ではなく、中身のファイルをアップロードしてください。index.html がリポジトリのルート（一番上の階層）にくるようにします。
4

GitHub Pages を有効化

  • リポジトリの Settings（設定）タブをクリック
  • 左メニューの Pages をクリック
  • Source: 「Deploy from a branch」 を選択
  • Branch: main、フォルダは / (root) を選択
  • Save をクリック
5

公開確認（1〜2分待つ）

  • ブラウザで以下のURLにアクセス:
https://あなたのユーザー名.github.io/pretzel-website/
  • サイトが表示されれば成功です 🎉
💡 ポイント: 表示されない場合は、1〜2分待ってから再度リロードしてみてください。初回はデプロイに少し時間がかかります。

🌐 オプション: 独自ドメイン設定

独自ドメインでサイトを公開する（任意）

  • お名前.com 等でドメインを取得（例: pretzel-mgmt.com、年間約1,500円）
  • GitHubリポジトリの Settings → Pages → Custom domain にドメインを入力
  • ドメイン側のDNS設定で CNAMEレコード を追加:
CNAME   →   あなたのユーザー名.github.io
💡 ポイント: 独自ドメインは後からいつでも設定できます。まずは github.io のURLで公開して問題ありません。

🔧 カスタマイズが必要な箇所

項目 やること
1. ロゴ assets/images/ に logo.png を追加し、各ページのテキストロゴ部分を <img> タグに差し替え
2. OGP画像 assets/images/ogp.png を作成（1200×630px推奨）。SNSでシェアした時のプレビュー画像になります
3. LINE QRコード contact.html と launch.html のQRコードプレースホルダーを実際の画像に差し替え
4. X アカウント名 各ページの @pretzel_mgmt を実際のアカウント名に変更
5. お問い合わせフォーム 将来的に Formspree や Google フォームに接続可能
6. Google Analytics 各HTMLファイルの </head> 直前にトラッキングコードを貼り付け
